# 📊 Análise de Dados - Tips Dataset

## Objetivo
Analisar o comportamento de gorjetas em restaurantes.

## Ferramentas
- Python
- Pandas
- Matplotlib

📊 INSIGHTS:

O jantar apresenta maior faturamento médio que o almoço
Fumantes possuem ticket médio ligeiramente maior
Os maiores valores estão concentrados em dias específicos
